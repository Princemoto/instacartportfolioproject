# Instacart Portfolio Project

This project analyses Instacart online grocery shopping data to uncover customer behaviour patterns, product demand, and reorder cycles.

## Tools Used
- Databricks SQL
- Microsoft Excel
- Power BI

## Repository Contents
- `report/` - Final report in Word format
- `excel/` - Excel workbook with charts and KPI validation
- `dashboard/` - Power BI dashboard screenshot
- `sql/` - SQL queries used for analysis

## Overview
The project follows the Databricks Medallion Architecture for structured data processing. Analysis and KPIs were validated in Excel and visualised with Power BI for actionable insights.
